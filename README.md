Mr. Squidee POS System
---
It is a Software Engineering project that implements a modular Point-of-Sale solution built to streamline restaurant operations, inventory management, and financial reporting.

---

**Features**

Order Processing: Real-time checkout.

Inventory CRUD: Full control over products (Add, Edit, Delete).

Sales Analytics: Automated data visualization and report generation.

User Management: Secure registration, profile updates, and role-based access.

Modular UI: Clean, dedicated CSS/JS architecture for every module.

---

**Tech Stack**

Backend: PHP

Database: MySQL

Server: XAMPP (Apache)

Frontend: JavaScript (ES6), CSS3, HTML5
